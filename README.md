# Bionic Athlete

Application for fitness  management of athletes or non-athletes.
- Create export reports from .fit files (e.g. Garmin, Strava) or .csv files like Trainerroad exports for handing off to an AI for coaching
- Nutrition tracking (also included in AI handoff)
- Body metrics tracking (also included in AI handoff)
- Local data history for privacy
- Integration of Garmin Conect, Strava, TrainingPeaks etc. via webhost
